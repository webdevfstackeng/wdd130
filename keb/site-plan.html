Content

Home

The main objective of my web developer online student portfolio is to cumulatively compile my work, knowledge and skills as I progress through the pathway towards earning the required certificates that will eventually qualify me for the desired Applied Technology degree award. The final product of my portfolio website will showcase an integrated platform of my web development talent that will be hosted life on GitHub with links to my LinkedIn, Twitter and Facebook social media accounts. I consider this online portfolio initiative as the perfect way to publish and market my professional web design and development skills set to create opportunities for better employment or future self-employment.

Achievements

A brief outline of skills, work done and certificates still to be acquired towards becoming a full stack web developer who specializes in both front-end and back-end development will be found on this page. Now my recently acquired skills include Python programming and Web Fundamentals which cover front-end web design and developmentwith hypertext markup language (HTML) and cascading style sheets (CSS) using VS Code IDE, User Interface (UI) design using website wireframe mockup design tools available online. I am excitedly looking forward to acquiring more knowledge about modern web development techniques, technologies and tools as my study programme semesters unfold.
